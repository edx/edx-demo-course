Forums de discussion

Le forum de discussion pour chaque cours se trouve en haut chaque page de cours. Vous pouvez cependant trouver une sous-partie de cette discussion à l'intérieur du cours (voir ci-dessous), vous pouvez discuter avec d'autres étudiants dans ce contexte précis. Allez-y, soyez social ! Créez votre premier message dans ce cours de démonstration.

Essayez de trouver des messages avec une encoche verte. L'encoche verte signifie que que message a été reconnu par un membre de l'équipe ou un modérateur du forum comme un poste valable. Vous pouvez aussi activement voter pour un message. Les autres utilisateurs peuvent chercher les messages qui on été votés par un utilisateur. Cela a tendance a être très utile.
