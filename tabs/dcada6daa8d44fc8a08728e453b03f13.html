Syllabus

Many courses include a syllabus tab that includes an overview of the course and what is expected of you. 

This demonstration course shows examples of everything that instructors might include in an edX course, and an explanation of how to interact with those components. The demo course does not distribute a certificate, although the grading system is active so that you can see how the grading system works. 

Like most courses on edX, the courseware is organized into a schedule, which you can see when you access the course material. In this case, that schedule is only there in order to illustrate how normal classes are organized. It is NOT expected that you take an entire week to go through "Week 1" of this demo course. The illustration schedule is roughly ordered by how often you might come across aspects of the edX platform. 

This syllabus allows you to access the course material by topic, in case you are looking for something specific on how to use the edX platform. If you just want a basic description of how to take a course on edX, start by clicking on 'the basics', below.
